# figma-to-ios-ui

Public iOS design-to-code skill package for AI agents.

## Layout

- `figma-to-ios-ui/`
  - Main skill package.
- `AGENTS.md`
  - Repo-level cross-agent trigger and editing instructions.
- `CLAUDE.md`
  - Claude Code entrypoint that imports `AGENTS.md`.

## What the skill covers

- UIKit and XIB generation, audit, and repair
- SwiftUI generation, audit, and repair
- hybrid UIKit and SwiftUI adaptation
- shared UIKit component hardening
- post-implementation validation against design evidence
- project-specific UI pattern learning through the companion sub-skill
